BUILD PROMPT — BrightFit Landing Page (full rebuild)
====================================================

Use this prompt to rebuild the entire BrightFit marketing site from scratch.
Place all files inside: landingpage/

Copy app assets from:
  - BrightFit/assets/icon.png  →  landingpage/assets/icon.png
  - BrightFit/images/mascot.png  →  landingpage/assets/mascot.png


GOAL
----
Build a modern, responsive, self-contained marketing website for the BrightFit
fitness app. Single-page main site (index.html) with Support and Privacy Policy
sections embedded on the page. Also create standalone privacy-policy.html and
support.html pages (for direct URLs / App Store compliance). All pages share
consistent branding and navigation.


BRAND & DESIGN
--------------
Match the BrightFit mobile app welcome screen aesthetic.

Colors:
  - Primary gold:     #FFB800
  - Deep gold:        #D89200
  - Matte black:      #121212
  - Charcoal gray:    #444444
  - White:            #FFFFFF
  - Soft background:  #F7F7F7
  - Border gray:      #EAEAEA

Typography:
  - Font: Plus Jakarta Sans (Google Fonts)
  - Logo wordmark: "Bright" (black, bold 900) + "Fit" (gold, bold 900, italic)
  - Tagline: "STRENGTH. SIMPLICITY. SUCCESS." (uppercase, letter-spaced)

Design style:
  - Clean, bold, fitness-focused
  - Gold CTA buttons with soft gold shadow
  - Rounded corners (16px cards, 14px buttons)
  - Sticky nav with blur backdrop
  - Scroll-reveal fade-in animations (IntersectionObserver)
  - Floating mascot animation in hero
  - Fully responsive (mobile + desktop)
  - Nav links scroll horizontally on mobile
